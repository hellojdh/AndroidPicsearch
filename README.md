AndroidPicsearch
================
For WinterCoding(hellojdh)
Flickr API를 이용한 간단한 사진 검색 앱


Activity 구성
------------

1. IntroActivity\
SplashActivity(1.3s)
2. MainActivity\
사진 검색 및 목록 보여주기\
3. DetailActivity\
선택 사진 원본 및 다운로드

Library 구성
-----------

1. Glide
2. Okhttp3
